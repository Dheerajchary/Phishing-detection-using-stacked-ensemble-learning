## Download Dataset

Get it from Mendeley Data:

> G. Vrbancic, "Phishing websites dataset," Mendeley Data, vol. 1, 2020.
> https://data.mendeley.com/datasets/72ptz43s9v/1

## After Downloading

Place the file here and name it:
```
data/dataset_small.csv
```

## Description

- ~58,000 samples (phishing + legitimate URLs)
- 111 features extracted from URLs
- Target column: `phishing` (1 = phishing, 0 = legitimate)
